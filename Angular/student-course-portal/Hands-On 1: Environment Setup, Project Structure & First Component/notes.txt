1. angular.json
Contains the Angular project configuration such as build, serve, assets, styles, and build options.

2. tsconfig.json
Main TypeScript configuration file for the entire project.

3. tsconfig.app.json
TypeScript configuration specifically for the Angular application.

4. package.json
Contains project information, dependencies, and npm scripts.

5. src/main.ts
The entry point of the Angular application. It starts (bootstraps) the application.

6. src/app/app.config.ts
Configures the standalone Angular application, including providers and routing.

7. src/app/app.component.ts
The root component of the application.

8. src/index.html
The main HTML page where the Angular application is loaded.
